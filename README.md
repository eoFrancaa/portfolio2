# 🚀 Portfólio - Rafael de França

Este é meu portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e experiências na área de tecnologia.

## 🛠️ Tecnologias

- Vue.js 3
- Vite
- JavaScript
- HTML
- CSS

## ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/eoFrancaa/portfolio2.git
```

Entre na pasta:

```bash
cd portfolio2
```

Instale as dependências:

```bash
npm install
```

Execute:

```bash
npm run dev
```
